E - Signo [B]

Time limit: 1 second
Memory limit: 256 megabytes
Languages: C#,C++11
  • All submissions
  • Accepted submissions
  • My submissions
  • PDF
  • Submit

Sobre una lista A de números enteros se desean realizar las siguientes operaciones:

  1. Update(i): Multiplicar por -1 el i-ésimo elemento de la lista A.
  2. Query(i, j): Devolver sign(∏k∈[i, j]Ak).

Input

Línea 1: Un entero n, 1≤n≤105, que indica el número de elementos de la lista A.

Línea 2: n enteros separados por espacio Ai, −106≤Ai≤106, los elementos de la lista A.

Línea 3: Un entero q, 1≤q≤105, que indica la cantidad de operaciones a realizar sobre A.

Línea 4 ... q+3: La descripción de cada operación. El formato es "1 i" (sin las comillas) para Update(i) y "2 i j" (sin las comillas) para Query(i, j) donde 0≤i≤j<n.

Output

Una línea por cada operación de tipo 2 con el resultado de su evaluación.

Sample test(s)

Input
5 3 -100 88 0 -27 8 2 0 2 2 2 4 1 2 2 1 2 1 3 1 0 2 0 2 2 0 4
Output
-1 0 1 -1 0

Plantillas:
